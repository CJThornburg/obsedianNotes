run tests
`npx playwright test`

run specific test
`npx playwright test <pathToTest>`

run test and see browser:
`npx playwright test --headed`
	- default is headless 

select browser
`npx playwright test --browser=firefox`
	- `=all`
	- `=firefox`
	- `=chromium`
	- `=webkit`
